# 后端技术栈文档

## 核心技术

### Go 语言
- **版本**: Go 1.21+
- **选择理由**: 高性能、并发支持优秀、编译型语言、部署简单

### Gin Web 框架
- **版本**: v1.9+
- **特性**:
  - 高性能 HTTP 路由
  - 中间件支持
  - JSON 验证
  - 错误管理
  - 路由分组
- **使用场景**: RESTful API 开发、路由管理、中间件链

### PostgreSQL 数据库
- **版本**: PostgreSQL 15+
- **特性**:
  - ACID 事务支持
  - JSON/JSONB 字段类型（存储 Block 内容）
  - 全文搜索
  - pgvector 扩展（向量检索）
  - 丰富的索引类型
- **使用场景**: 主数据存储、关系数据管理、向量相似度搜索

### pgvector 扩展
- **版本**: v0.5+
- **特性**:
  - 向量数据类型支持
  - 余弦相似度、欧氏距离计算
  - HNSW 索引（高性能近似最近邻搜索）
  - IVFFlat 索引
- **使用场景**: 
  - 语义搜索（将文本转为向量后检索）
  - 相似内容推荐
  - AI 驱动的内容发现

## 数据访问层

### GORM
- **版本**: v1.25+
- **特性**:
  - ORM 映射
  - 自动迁移
  - 关联关系处理
  - 事务支持
  - 钩子函数
- **使用场景**: 数据库操作、模型定义、查询构建

## 缓存层

### Redis
- **版本**: Redis 7+
- **特性**:
  - 内存存储，极速读写
  - 多种数据结构（String、Hash、Set 等）
  - 过期时间设置
  - 发布订阅
- **使用场景**:
  - 已发布文章的 Block 数据缓存
  - Session 存储
  - 热点数据缓存
  - 缓存失效策略

### go-redis 客户端
- **版本**: v9+
- **特性**: Redis 协议完整实现、连接池、Pipeline 支持

## 认证与安全

### JWT (JSON Web Token)
- **库**: golang-jwt/jwt v5+
- **用途**: 
  - 无状态认证
  - Token 生成与验证
  - Claims 自定义
- **实现**: 
  - Access Token（短期有效）
  - Refresh Token（长期有效，可选）

### bcrypt
- **库**: golang.org/x/crypto/bcrypt
- **用途**: 密码哈希与验证
- **特性**: 自适应哈希算法、防彩虹表攻击

## 配置管理

### Viper
- **版本**: v1.17+
- **特性**:
  - 多格式配置文件支持（YAML、JSON、ENV）
  - 环境变量读取
  - 配置热加载

## 日志

### zap

## 数据验证

### validator
- **库**: go-playground/validator v10+
- **特性**:
  - 结构体字段验证
  - 自定义验证规则
  - 错误信息本地化
- **使用场景**: API 请求参数验证、数据完整性检查

## 向量化与 AI 集成（可选）

### OpenAI API / 本地模型
- **用途**: 将文本内容转换为向量嵌入
- **流程**:
  1. 用户保存 Block 内容
  2. 后端调用嵌入模型生成向量
  3. 存储向量到 PostgreSQL（pgvector 字段）
  4. 用户搜索时，查询向量相似度

### 可选方案
- **OpenAI Embeddings API**: text-embedding-3-small
- **本地模型**: sentence-transformers（通过 Python 微服务）
- **Ollama**: 本地部署的开源模型

## 技术选型总结

| 技术 | 用途 | 优势 |
|------|------|------|
| Go + Gin | Web 服务 | 高性能、简洁、并发友好 |
| PostgreSQL | 数据存储 | 功能强大、支持 JSON 和向量 |
| pgvector | 向量检索 | 原生支持、性能优秀 |
| Redis | 缓存 | 极速读写、丰富数据结构 |
| JWT | 认证 | 无状态、易扩展 |
| GORM | ORM | 开发效率高、功能完善 |
