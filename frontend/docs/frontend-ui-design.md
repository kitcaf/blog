这几张 UI 图非常完美地呈现了我们之前讨论的**“极客命令枢纽 (Command Hub) + 经典分栏账本 (Split Ledger)”**的融合设计。

这种风格去除了所有工业流水线化的“网页感”（没有卡片、没有阴影、没有色块填充），完全靠**空间留白**和**文字排版**来撑起高级感。

根据你上传的三张设计图（主页、列表页、详情页），我为你提炼出这套博客系统的核心 UI/UX 规范，你可以直接把它作为未来开发的“设计原则 (Design System)”：

### 1. 整体视觉风格 (Visual Aesthetic)

* **极致留白 (Whitespace as UI)**：抛弃所有背景色和容器边框。整个屏幕就是一张无限大的白纸。
* **去卡片化 (Cardless Design)**：彻底消灭了传统博客的“大方块卡片”。信息不再被包裹在盒子里，而是直接“呼吸”在画布上。
* **排版驱动 (Typography-Driven)**：因为没有图形和色块，视觉层级完全依赖文字的**大小、粗细（字重）、颜色深浅（黑 vs 浅灰）**以及**对齐方式**来建立。

### 2. 核心排版规范 (Typography Rules)

* **色彩克制**：全局几乎只有三种颜色：
* `#000000` (或极深的灰 `#111827`)：用于当前选中项、文章大标题、正文核心文字。
* `#9CA3AF` (或类似浅灰色)：用于所有次要信息（日期、标签、未选中的导航栏、搜索框占位符）。
* `#FFFFFF`：纯白背景。


* **字体组合**：
* **大标题/列表标题**：使用了非常优雅的衬线字体 (Serif) 或极具现代感的无衬线字体，字号大，视觉冲击力强（如列表中 `The Evolution of React...`）。
* **元数据 (Metadata)**：日期 (`10.24`)、标签 (`REACT, RSC`) 以及表头 (`DATE`, `TITLE`, `TAGS`) 使用了极小字号、全大写 (All-caps) 的无衬线或等宽字体，营造出一种“严谨的数据账本”质感。



### 3. 三大核心页面布局 (Layout Details)

* **Home 主页 (The Portal)**：
* 绝对居中对齐。
* 只有一个巨大的带极浅下划线的输入框 (`Search or command...`)。
* 下方紧跟一行浅灰色的极简分类导航，没有任何多余干扰。


* **Blog 列表页 (The Split Ledger)**：
* **顶部 Header**：主页的搜索框缩小并移动到了上方，右侧是主导航 (`Blog`, `Project`, `Me`) 和社交图标。
* **左侧导航 (侧边栏)**：当前选中项 (`Timeline`) 用**纯黑加粗**并在左侧加了一个极其克制的**小黑点 (`•`)**，未选中项保持浅灰。
* **右侧账本区**：绝美的多列表格排版。顶部有小字号表头。每篇文章是一行，左边对齐日期，中间对齐标题，右边对齐标签。行与行之间只有极其微弱的 `1px` 浅色底边框 (`border-b`)。


* **Article 详情页 (The Canvas)**：
* 单栏居中阅读布局，控制了最大阅读宽度（通常是 `max-w-2xl` 或 `65ch`），保证眼睛阅读不疲劳。
* 顶部保留搜索栏，左侧出现一个纯文字的 `← Back to Index` 返回按钮。
* 正文上方的元数据以 `10-24 // Backend` 这种极具极客感的双斜杠形式呈现。正文行距非常宽松 (`leading-relaxed`)。

### 4. 关键动效设计 (Animation & Transitions - 结合 Vue 架构)

虽然图是静态的，但根据这套 UI 的逻辑，它的动效（如我们前面讨论的 View Transitions）必须遵循以下原则：

* **主角的形变 (Morphing)**：那个带有 `Search or command...` 的输入框是唯一的视觉锚点。当从“主页”点击进入“列表页”或“详情页”时，这个搜索框**绝不能消失再重新渲染**，而是应该平滑地缩小尺寸，并像滑行一样从屏幕中央移动到屏幕顶部。
* **淡入淡出 (Fade & Slide)**：
* 主页下方的菜单在跳转时平滑淡出。
* 列表页的“左侧导航”和“右侧账本”应该有一个非常轻微的错幅淡入动画（比如从下往上偏移 `10px` 同时透明度从 0 到 1）。


* **悬停微交互 (Hover States)**：
* 导航栏和列表行悬停时，不需要夸张的背景色变化。只需要文字颜色从浅灰变为深黑，或者整行出现极度难以察觉的极浅灰色背景 (`hover:bg-gray-50`) 即可。
