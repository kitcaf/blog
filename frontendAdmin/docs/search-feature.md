# 搜索功能文档

## 功能概述

全局搜索功能允许用户快速查找页面内容，支持实时搜索和键盘导航。

## 使用方式

### 1. 打开搜索对话框

- **快捷键**：`Ctrl + P` (Windows/Linux) 或 `Cmd + P` (Mac)
- **点击按钮**：点击侧边栏的"搜索"按钮

### 2. 搜索操作

- 输入关键词自动搜索（300ms 防抖）
- 按 `↑` `↓` 键导航结果
- 按 `Enter` 键打开选中的页面
- 按 `Esc` 键关闭搜索对话框

### 3. 搜索结果

结果按时间分组显示：
- 今天
- 上周
- 过去 30 天
- 更早

每个结果显示：
- 页面图标和标题
- 页面路径（面包屑）
- 匹配的内容预览（最多 3 个 Block）

## 技术实现

### 组件结构

```
SearchDialog (主组件)
├── 搜索输入框
├── 加载状态
└── SearchResultGroup (结果分组)
    └── 结果列表
```

### 核心 Hooks

1. **useSearchQuery** - 搜索查询
   - 使用 React Query 管理搜索请求
   - 自动缓存搜索结果（30 秒）
   - 只在有查询内容时发起请求

2. **useDebounce** - 防抖
   - 延迟 300ms 发起搜索请求
   - 避免频繁请求后端

3. **useGlobalShortcut** - 全局快捷键
   - 监听 `Ctrl + P` 快捷键
   - 自动处理 Mac 的 `Cmd` 键

### 性能优化

1. **防抖搜索**：输入停止 300ms 后才发起请求
2. **结果缓存**：相同查询 30 秒内不重复请求
3. **按需渲染**：只在对话框打开时渲染组件
4. **虚拟滚动**：结果列表支持大量数据（未来优化）

### 状态管理

- **本地状态**：搜索对话框开关、输入内容、选中索引
- **React Query**：搜索结果缓存和加载状态
- **无全局状态**：避免不必要的重渲染

## 样式设计

### 颜色系统

使用 CSS 变量保证深色模式兼容：

- `--color-app-bg` - 背景色
- `--color-app-fg` - 前景色
- `--color-app-fg-light` - 浅色文字
- `--color-app-fg-deeper` - 深色文字
- `--color-app-hover` - 悬停背景

### 动画效果

- **淡入**：对话框背景 200ms fade-in
- **缩放**：对话框内容 200ms zoom-in-95
- **平滑滚动**：选中项自动滚动到可见区域

## 键盘导航

| 按键 | 功能 |
|------|------|
| `Ctrl + P` | 打开搜索对话框 |
| `↑` | 上一个结果 |
| `↓` | 下一个结果 |
| `Enter` | 打开选中的页面 |
| `Esc` | 关闭对话框 |

## 未来优化

1. **搜索历史**：记录最近搜索的关键词
2. **搜索建议**：根据输入提供关键词建议
3. **高级筛选**：按类型、标签、日期筛选
4. **虚拟滚动**：优化大量结果的渲染性能
5. **搜索高亮**：在结果中高亮匹配的关键词
6. **快速预览**：悬停结果时显示页面预览

## 错误处理

1. **网络错误**：显示"搜索失败，请稍后重试"
2. **空结果**：显示"未找到相关内容"
3. **空查询**：显示"输入关键词开始搜索"
4. **加载状态**：显示加载动画
